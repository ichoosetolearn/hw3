# HW Books Parser

## Description
Gathering info about books from [toscrape.com](https://toscrape.com/)

## Usage
scrape_books() - to scrape all books, without saving the result to a file.  
scrape_books(True, 5) - to scrape 5 books and save information to books_data.txt.

## Used Libs
[requests](https://requests.readthedocs.io/en/latest/)  
[BeautifulSoup](https://www.crummy.com/software/BeautifulSoup/)
